##DP动态规划的使用条件：
1最优子结构性质。如果一个问题的最优解包含的子问题解也是最优的，即为最优子结构
2无后效性。父可以推子，但是子的结果不能对父有任何影响
3.子问题重叠性：子问题重叠性是指在递归算法自顶向下进行求解时，子问题并不总是新问题，每一个子问题只计算一次，并将其存在表格中。
当需要的时候，只用去查看表格的内容即可
